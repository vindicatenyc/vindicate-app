# YouTube Long-Form Video Script Template

**Duration:** 8-12 minutes
**Aspect Ratio:** 16:9 (horizontal)

## Structure

### 1. HOOK (0:00 - 0:30)
Goal: Stop the scroll, promise value
- Start with the problem
- Tease the solution
- "By the end of this video, you'll know..."

### 2. INTRO (0:30 - 1:30)
Goal: Establish credibility and context
- Who we are (Vindicate NYC)
- Why we're building this
- What's in this video

### 3. THE PROBLEM (1:30 - 3:00)
Goal: Make viewers feel understood
- Paint the picture of the struggle
- Statistics that validate the problem
- "If you're dealing with X, you're not alone"

### 4. THE SOLUTION (3:00 - 7:00)
Goal: Teach and demonstrate
- Explain the concept/feature
- Show it in action (screen share)
- Step-by-step walkthrough
- Address common questions

### 5. DEEP DIVE (7:00 - 9:00)
Goal: Provide advanced value
- Technical details for those who want them
- Edge cases and considerations
- Pro tips

### 6. NEXT STEPS (9:00 - 10:00)
Goal: Actionable takeaways
- What to do right now
- Resources mentioned
- Upcoming features

### 7. CTA (10:00 - 10:30)
Goal: Engagement and growth
- Subscribe and notification
- Comment prompt (engagement)
- Link to try the tool
